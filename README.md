# ClientVault - Document Management System

Hi! This is my full-stack project made for internship submission.
ClientVault is a simple web application that helps a company manage client documents and family information in one place.
It allows users to upload, view, replace, and delete documents like Aadhar, PAN, Driving License, and Electricity Bill. Documents are managed person-wise, making it easy to organize and access data.
The system also includes a family relationship feature where you can view a person's family structure including head of family, spouse, and children in a visual format.

## Project Structure
backend - Spring Boot application handling APIs and database  
frontend - React application for user interface

## Tech Stack
Frontend: React, JavaScript, Axios  
Backend: Java, Spring Boot, REST APIs  
Database: MySQL

## How to run
Backend:
cd backend  
./mvnw spring-boot:run  

Frontend:
cd frontend  
npm install  
npm start
